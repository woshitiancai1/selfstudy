Node.js不是一门语言、不是库、不是框架、
Node.js是一个JS运行时环境
Node.js可以解析和执行JS代码
Node.js中的JS：没有DOM、和BOM，服务端不操作页面
    在Node这个JS执行环境中为JS提供了一些服务器级别的操作API：
    例如：文件读写、网络服务的搭建、网络通信、http服务器
    

Node.js（特性）：事件驱动、非阻塞IO模型(异步)、轻量和高级

npm是世界上最大的开源库生态系统
绝大多数JS相关的包都存放在npm上，这样做的目的是让开发人员更方便的下载使用
npm install
